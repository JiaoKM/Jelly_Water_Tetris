# Jelly_Water_Tetris

When the number of blocks in one row reaches a pre-set value, the material of blocks will convert from jelly to water.

![tetris](./image/Tetris.png)

## Demo

![demo](./image/video.gif)

## Usage

```bash
python jelly_water_tetris.py
```

Use **P** key to pause.
